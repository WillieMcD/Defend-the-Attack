Defend the Attack - Web Game

How to run:
1. Open index.html in a modern browser.
2. Choose Two Player or Single Player vs AI.
3. Pick a column, answer the question, and connect four to win.

Staff question spreadsheet upload:
- The game can replace the built-in cyber security questions with a staff-created CSV spreadsheet.
- Open questions_template.csv in Excel or Google Sheets.
- Keep the column headings exactly as they are:
  Category, Question, Option A, Option B, Option C, Option D, Correct Answer, Explanation
- Correct Answer can be A, B, C, D, 1, 2, 3, 4, or the exact text of the correct option.
- Save/export the file as CSV UTF-8.
- In the game, use "Upload question spreadsheet".

Best practice:
- Use exactly 7 categories if you want each Connect 4 column to have a unique topic.
- Add at least 7 valid questions. More is better for replay value.
- If fewer than 7 categories are supplied, the game fills the remaining columns with mixed questions.
